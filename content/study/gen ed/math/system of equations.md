
## meaning
systems of equations are just two (or more) math statements that need to be true at the same time. We're looking for values that make all equations work.

## example
3x - 2y = 7 
4x + 5y = 1

1. isolate y
	- move 3x to the right
		- -2y = 7 - 3x
	- divide by -2
		- y = (-3x - 7)/2
2. replace 7 in the other equation
	- 4x + 5((-3x - 7)/2) = 1
3. simplify parentheses
	- 5 x (3x - 7)/2
	- (15x - 35)/2
	- 4x + (15x - 35)/2 = 1
		- times 4x by 2 
	- (8x + 15x - 35)/2 = 1
	- (23x - 35)/2 = 1
	- 23x - 35 = 2
		- add 35 to both sides
	- 23x = 37
	- x = 37/23
4. putting x back
	- y = (3(37/23) - 7)/2)
		- only distribute to the first one (numerator)
	- y = (111/23 - 7)/2
	- y = (111/23 161/23)
	- y = (-50/23)/2

y = -25/23
