## pre production
- script breakdown
    - process of identifying all a scripts elements to help prep for shooting and scheduling
    - done by producer or 1st assistant director 
    - break up each script page into 8 parts
        - estimates the amount of time needed per scene
    - colour code different types of script elements 
        - characters
        - sfx
        - props
        - costuming
        - makeup 
        - vfx
        - extras
        - vehicles
        - animals
        - children
        - special equipment
        - set dressing
    - do for every scene
